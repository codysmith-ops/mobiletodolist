Spacing & Sizing Standards
Border Standards
1px - All borders (cards, inputs, badges)
2px dashed - Upload areas, placeholders only
Border Radius
6px - Badges, pills
8px - Buttons, small cards
12px - Cards, containers
Box Shadows
Light: 0 2px 8px rgba(15, 23, 42, 0.04)
Elevated: 0 4px 16px rgba(15, 23, 42, 0.15)
Featured: 0 8px 32px rgba(81, 89, 176, 0.2)
Key Design Rules
✅ Borders: Always 1px (never 2px or 3px) - use shadows for depth
✅ Border Radius: 6px (badges), 8px (buttons), 12px (cards)
✅ Dark Backgrounds: Use #1E293B (slate), NOT #0F172A (pure black)
✅ Text on Dark: #F1F5F9 (primary) → #94A3B8 (secondary) → #64748B (tertiary)
✅ Labels on Dark: Always #818CF8 (light indigo)
✅ Alert Colors: Red/Yellow/Green ONLY for notifications, not general UI
✅ Headings: Montserrat Bold, Body: Inter Regular, Code: Monospace
